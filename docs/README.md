# Cursor Self-Host Planning Documentation

This directory contains structured planning documents for the self-hosted Cursor CLI wrapper project. Each document can be refined independently.

| Document | Purpose |

| [03-design.md](./03-design.md) | Design decisions, UX principles, and visual direction |
| [04-technical-decisions.md](./04-technical-decisions.md) | Architecture, tech stack, and implementation choices |

## How to Use

1. **Refine iteratively** — Each doc is a living document; update as decisions solidify.
2. **Cross-reference** — Technical decisions inform layouts; layouts inform design.
3. **Track status** — Use checkboxes and status markers in each doc to track progress.
